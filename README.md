## Quick start
There is no build step or installation.
 
- **Just want to view it?** Open the live link above.
- **Want to run it locally?** Clone the repo and open `index.html` in your browser:
```bash
git clone https://github.com/Nam2302/naman-site.git
cd naman-site
open index.html   # or just double-click it
```
 
## Features
- Clean, minimal layout with a graph-paper style background
- About section covering who I am and what I'm into
- Project write-ups for things I've actually built and shipped, with links to try them
- Contact section linking out to my GitHub
- No frameworks or build tools

**Files:**
```
index.html   -> page structure/content
style.css    -> all styling
```
 
## How it works
The layout is built with plain flexbox rather than CSS Grid, kept intentionally simple so the whole stylesheet is easy to read top to bottom. Colors, fonts, and spacing are defined once as CSS custom properties (`:root` variables) so the whole site's look can be changed by editing a handful of values instead of hunting through the file.
